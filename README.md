# zdy.blog.core

使用 ASP.NET Core 2.0 编写，其中一些代码借鉴了 [Miniblog.Core](https://github.com/madskristensen/Miniblog.Core) ，开发了后台，加入了个人喜欢的 Markdown 来编辑博客。

## 使用的库

- [CommonMark.NET](https://github.com/Knagis/CommonMark.NET) 
- [Pomelo.EntityFrameworkCore.MySql](https://github.com/PomeloFoundation/Pomelo.EntityFrameworkCore.MySql)
- [SixLabors.ImageSharp](https://github.com/SixLabors/ImageSharp)
- [WebMarkupMin.AspNetCore2](https://github.com/Taritsyn/WebMarkupMin)

## 前端插件

- jquery
- jquery-fancybox
- jquery-lazyload
- marked
- prism
